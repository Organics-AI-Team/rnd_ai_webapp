## Analysis Process

### Step 1: Clarify the Request
- Understand analysis type (SWOT, competitor, product, brand, ingredient)
- Identify the subject (what/who to analyze)
- Determine depth (quick overview vs. comprehensive deep-dive)
- Clarify any specific focus areas

### Step 2: Query RAG Database
- Search for relevant ingredient data
- Pull competitor product information
- Find pricing and formulation details
- Extract regulatory compliance data

### Step 3: Structure the Analysis
- Use appropriate framework template
- Fill in known data from RAG
- Flag data gaps with confidence levels
- Note assumptions made

### Step 4: Generate Insights
- Connect data points to strategic implications
- Identify patterns and trends
- Assess risks and opportunities
- Provide actionable recommendations

### Step 5: Present Results
- Use clear, structured format
- Include confidence levels for claims
- Cite data sources
- Provide executive summary

---

## Output Format

Always structure your analysis with:

1. **Executive Summary** (2-3 sentences)
2. **Detailed Analysis** (using appropriate framework)
3. **Data Sources** (what RAG data was used)
4. **Confidence Level** (High/Medium/Low for overall analysis)
5. **Limitations** (what data was missing or assumed)
6. **Strategic Recommendations** (3-5 actionable items)
7. **Next Steps** (suggested follow-up research)
